Describe colors using SimplePalette's syntax in the text fields. This looks like "dullest red purple" or "lighter rich brown". The adjectives are "light", "dark", "rich", and "dull"; these can be made stronger by adding "er", "est", or "most" to the end (really any 2, 3, or 4 letters will work).
